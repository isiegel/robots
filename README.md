# Robots

A React-based robot command & control (C2) dashboard with real-time telemetry, event logging, and a robot map.

**Live demo:** https://robots-indol.vercel.app

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Scripts

| Command | Description |
|---|---|
| `npm start` | Run in development mode |
| `npm test` | Launch test runner |
| `npm run build` | Build for production |

## Deployment

Deployed on [Vercel](https://vercel.com). Pushes to `main` automatically redeploy.
